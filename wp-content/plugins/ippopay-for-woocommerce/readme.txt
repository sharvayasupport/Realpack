Ippopay Payments plugin to accept credit card, UPI, Netbanking and debit card payments.

== Description ==

The Ippopay Payments plugin allows you to accept credit card, UPI, Netbanking and debit card payments via Ippopay payment gateway on your WordPress site easily.
This plugin will add a Ippopay Payment option on WooCommerce checkout page, when user choses Ippopay as Payment Method, he will have redirected to Ippopay popup to complete his transaction Ippopay will send that user back to your website along with transactions details.

One click payment via Ippopay with a remember me feature. Responsive design so it is compatible with all devices and browsers.


== Installation ==

In the WordPress administration panel:

1. Go to WooCommerce -> Settings section->Payments

2. You can see Ippopay. Click manage button

3. Add Ippopay merchant key and Save (You can take this merchant key from Ippopay merchant panel under API Setting tab

If you have any questions or issues, feel free to contact our technical support: info@Ippopay.com

== Dependencies == 

1. Wordpress v3.9.2 and later
2. Woocommerce v2.4 and later
3. PHP v5.6.0 and later
4. php-curl extension

Yes! In live mode, an SSL certificate must be installed on your site to use Ippopay Payment Gateway.

Yes. Please get multi-currency enabled for your account. Once you have it enabled, you can install this plugin
